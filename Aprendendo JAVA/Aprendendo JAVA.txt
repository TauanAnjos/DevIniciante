Tecnica CamelCase(tecnica que parece corcovas do camelo)
ex --> TauanAnjosNascimento = São 3 palavras, sempre que a primeira palavra estiver em maiúscula, ela representa uma classe/interface

Se a primeira letra for "Minúscula" ela pode ser = um atributo, uma variavel ou um metodo.

Ex Atributo = nomeAluno

Ex Variável = mediaPrimeiroBimestre

Ex Método = lancarNotaAluno

Nesses exemplos acima, vimos que a primeira letrasé sempre minúsculas, e o resto usa o camelcase.
Mas se encontrar a palavra com todas letras minúsculas, ela é nome do pacote.
Ex nome do pacote: meu_pacote = tauananjosnascimento

Já se encontrar todos os nomes em letras maiúsculas, ela é uma constante.
Ex constante: MINHA_CONSTANTE = VALOR_DE_PI , que é sempre constante.

relogio = New --> quando vc coloca o "New " nessa situação, quer dizer que vc está criando novo objeto.
